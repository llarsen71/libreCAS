LibreCAS
--------

This project is intended to work with LibreOffice as the basis for developing a CAS
structure.  Blocks of code can be included in the LibreOffice file and executed.
Sympy will be the algebra backend.  This was originally developed in OpenOffice 3.2
and may need to be updated for LibreOffice

The python script files can be placed in the folder:

<LibreOffice>/Basis/share/Scripts/python
